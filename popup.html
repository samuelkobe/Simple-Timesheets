Google Drive Sync

Drive sync stores your data in a private folder on your Google Drive — invisible in the Drive UI, accessible from any browser you're signed into.

1 Create a project in Google Cloud Console and enable the Drive API + People API
2 Create an OAuth 2.0 Client ID (type: Chrome Extension) and add this extension's ID
3 Paste the Client ID into manifest.json and reload the extension
